# Keep mockup

googleのkeep（メモ）サービスのモックアップ

# DEMO
 [youtube demo](https://youtu.be/6ewOzS07fzA)

# Features
 
メモを残せます
 
# Requirement

* React 18.0.0
* MUI 5.6.2
 
# Installation

ローカル開発ではdocker desktopが必要
プロジェクトルートディレクトリで、以下のコマンドを実行
dockerコンテナが立ち上がります
```bash
docker-compose up -d
```
